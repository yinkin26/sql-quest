üìå What is SELECT?

The SELECT statement is used to fetch data from a table in a database. It's the most commonly used SQL command!

üß† Basic Syntax

To retrieve data from all columns of a table:

SELECT * FROM table_name;

To get specific columns:

SELECT column1, column2 FROM table_name;

Example:

SELECT name, email FROM customers;

üß™ Try It Yourself

üìä Select Specific Columns

If you only want names and cities:

SELECT name, city FROM customers;

üîç Filtering Results with WHERE

Use the WHERE clause to filter rows based on conditions:

SELECT name, city FROM customers WHERE city = 'London';

Example: See customers who live in Tokyo

üì¶ Select from Another Table

If you have more than one table, you can fetch data from any:

SELECT * FROM orders;

üí¨ Behind the Scenes

The SELECT statement sends a request to the database asking it to return specific rows and columns from a table. If a WHERE clause is included, only rows that match the condition are returned.

üí° Helpful Tips

  • * selects all columns, but it's better to name them explicitly for clarity and performance.
  • Use single quotes ' ' for text values in conditions.
  • SQL keywords like SELECT, FROM, and WHERE are not case-sensitive.
  • SELECT only reads data; it does not change the database.
  • End your queries with a semicolon ; especially when writing multiple queries.

üß† Summary

  • SELECT reads data from a table
  • * selects all columns; use column names for specific ones
  • FROM tells SQL which table to query
  • WHERE filters the rows returned

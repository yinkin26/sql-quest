üîç What is WHERE?

The WHERE clause is used in SQL to filter records and return only those that meet a specified condition.

üß† Basic Syntax

SELECT column1, column2 FROM table_name
WHERE condition;

üí° Example

SELECT * FROM customers
WHERE city = 'Tokyo';

‚öôÔ∏è Common Operators with WHERE

  • = (equal)

  • >, <, >=, <= (greater/less than)
  • LIKE (pattern match)
  • IN (match any from a list)
  • BETWEEN (between two values)

üåç Real-World Use Case

You want to find customers from cities in Japan. Assuming Tokyo and Osaka are the options:

SELECT name FROM customers
WHERE city IN ('Tokyo', 'Osaka');

üí° Helpful Tips

  • WHERE is case-insensitive in most databases, but ensure values match the case stored in the database for accuracy.
  • You can combine conditions with AND / OR operators.
  • Use 'value' for string comparisons and no quotes for numbers.

üß† Summary

  • WHERE filters rows returned by a SELECT, UPDATE, or DELETE query.
  • Use comparison operators like =, <>, >, etc.
  • Combine multiple conditions with AND or OR.

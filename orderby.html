üìå What is ORDER BY?

The ORDER BY clause is used to sort the result set in either ascending or descending order.

üß† Basic Syntax

SELECT column1, column2 FROM table_name
ORDER BY column1 [ASC|DESC];

Note: If you don't specify ASC or DESC, the default is ASC (ascending order).

üìä Sorting by Multiple Columns

SELECT name, city, age FROM customers ORDER BY city ASC, age DESC;

If multiple customers are from the same city, the second column (age) determines their order within that group.

üß™ Try It Yourself

üåç Real-World Use Case

Imagine you are analyzing customer data for a company and want to display the list of customers sorted by their age. To quickly analyze the youngest and oldest customers, use:

SELECT name, age FROM customers ORDER BY age DESC;

This allows you to instantly view the oldest customers at the top of the list, which can be useful for targeted promotions or research.

üí° Helpful Tips

  • ORDER BY is used to sort results by one or more columns.
  • If you don't specify ASC or DESC, ASC (ascending) is the default.
  • DESC is used for descending order (from highest to lowest).
  • You can sort by multiple columns, with different sort directions for each column.

üß† Summary

  • ORDER BY is used to sort results by one or more columns.
  • ASC is the default sort direction (from lowest to highest).
  • DESC is used for descending order (from highest to lowest).
  • You can sort by multiple columns, with different sort directions.

üìå What is SELECT DISTINCT?

The SELECT DISTINCT statement is used to return only unique (non-duplicate) values from a column in a table.

üß† Basic Syntax

You can use SELECT DISTINCT to get unique values from a single column:

SELECT DISTINCT column1 FROM table_name;

You can also select multiple columns:

SELECT DISTINCT column1, column2 FROM table_name;

üí° Example

SELECT DISTINCT city FROM customers;

üß™ Try It Yourself

üí¨ Why Use DISTINCT?

Sometimes tables have repeated values in columns. If you want to find out which different values exist, DISTINCT helps filter out duplicates.

For example, if multiple customers live in the same city, SELECT city will list the same city multiple times, but SELECT DISTINCT city will only show each city once.

üåç Real-World Use Case

Imagine you're analyzing a customer list and want to know how many different cities your customers come from. Instead of counting all rows, use:

SELECT COUNT(DISTINCT city) FROM customers;

This will return the number of unique cities only.

üí° Helpful Tips

  • DISTINCT applies to all columns listed after it.
  • If you SELECT DISTINCT with multiple columns, rows must be unique across all listed columns to appear.
  • Can be used with ORDER BY to sort results.
  • Use DISTINCT early in analysis to quickly explore available values in a column.

üìå What is the CREATE TABLE Statement?

The CREATE TABLE statement is used to create a new table in a database. You define the table structure, including column names and data types, so the database knows how to store the data.

üß† Basic Syntax

CREATE TABLE table_name (
  column1 datatype,
  column2 datatype,
  column3 datatype
);

üí° Example (User Table)

CREATE TABLE users (
  user_id INT PRIMARY KEY,
  username VARCHAR(100),
  email VARCHAR(100),
  created_at DATETIME
);

üåç Real-World Use Case

SQL CREATE TABLE is commonly used in web applications to structure data. For example, an e-commerce platform might use the CREATE TABLE statement to define a products table with columns like product_name, price, and quantity.

CREATE TABLE products (
  product_id INT PRIMARY KEY,
  product_name VARCHAR(100),
  price DECIMAL(8, 2),
  quantity INT
);

üí° Helpful Tips

  • Use PRIMARY KEY to uniquely identify records in the table.
  • VARCHAR is great for storing text, while INT and DECIMAL are ideal for numeric data.
  • Don‚Äôt forget to specify column data types accurately based on the type of data you intend to store.

üß† Summary

  • CREATE TABLE is used to create a new table in the database.
  • Column names and data types are defined to specify the structure of the table.
  • PRIMARY KEY ensures that each row in the table is unique.
